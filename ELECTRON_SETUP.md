# Инструкция по сборке Electron приложения

## Шаг 1: Установка зависимостей

Убедитесь, что у вас установлен Node.js (версия 14 или выше).

Откройте терминал в папке проекта и выполните:

```bash
# Инициализация (уже выполнено, package.json создан)
# Установка Electron и упаковщика
npm install
```

## Шаг 2: Тестирование приложения

Проверьте, что приложение работает:

```bash
npm start
```

Должно открыться окно приложения. Если всё работает, переходите к следующему шагу.

## Шаг 3: Сборка EXE файла

Для Windows:

```bash
npm run build
```

Или для всех платформ:

```bash
npm run build-all
```

## Результат

После сборки в папке проекта появится папка `BeadingPatternEditor-win32-x64` (или другая, в зависимости от платформы).

Внутри будет:
- `BeadingPatternEditor.exe` - исполняемый файл
- Все необходимые файлы приложения

## Распространение

1. Заархивируйте всю папку `BeadingPatternEditor-win32-x64`
2. Отправьте архив пользователю
3. Пользователь распаковывает и запускает `.exe` файл

## Дополнительные настройки

### Добавление иконки

1. Создайте файл `icon.ico` (для Windows) или `icon.png` (для других платформ)
2. Раскомментируйте строку с `icon` в `electron-main.js`
3. Пересоберите приложение

### Изменение размеров окна

Отредактируйте `electron-main.js`:
```javascript
width: 1400,  // Измените на нужную ширину
height: 900,  // Измените на нужную высоту
```

### Включение DevTools для отладки

В `electron-main.js` раскомментируйте:
```javascript
win.webContents.openDevTools()
```

## Устранение проблем

### Ошибка "Cannot find module 'electron'"
Выполните: `npm install`

### Приложение не открывается
Проверьте консоль на наличие ошибок: `npm start`

### EXE файл не запускается
Убедитесь, что все файлы проекта (включая `Sketch_base.dxf`) находятся в папке сборки
